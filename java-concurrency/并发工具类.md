# 并发工具类

1、闭锁-CountDownLatch

允许一个或多个线程等待其他线程完成操作

闭锁的常见两种用法

- 控制多个线程同时开始，并当多个线程结束时，继续主线程。

- 闭锁是一次性对象，一旦进入终止状态，就不能被阻塞。




2、同步屏障-CyclicBarrier

让一组线程达到一个屏障（同步点）时被阻塞，直到最后一个线程到达屏障时，屏障才会开门，所有被屏障拦截的线程才会继续进行

3、信号量-Semaphore

控制并发的数量-用来控制访问某个特定资源的操作数量，或者同时执行某个操作的数量



应用场景: 流量控制



4、交换者-Exchanger

用于进行线程间的数据交换-提供一个同步点，在这个同步点两个线程可以交换彼此的数据。
