单核的情况下

多线程的本质，应该解释为叫多任务，从始至终，只有一个人在执行任务。只不过说在这些任务之间交替执行，给人的感觉就好像多个人在同时执行任务，多线程应该叫做像多线程，是一种假象。

一个人好像干了多个人的活



多核的情况下

为什么会出现线程安全，原因就是，多个任务在做同一件事，做这件写要使用相同的工具

3. 如何避免死锁？

   - 避免一个线程同时获得多个锁
   - 避免一个线程同时占用多个资源，尽量保证每个锁只占用一个资源
   - 使用定时锁
   - 数据库锁，加锁，解锁必须在一个数据库里
